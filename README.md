# rbhoodlanding

Premium landing page for Robinhood positioned as an independent educational analytics platform for traders.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
